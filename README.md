# 📦 Installer Creator

## 🚀 Создатель профессиональных установщиков на Python

Простой, но мощный инструмент для создания установщиков Windows с красивым GUI, автоматическим запросом прав администратора и надёжным созданием ярлыков.

---

## ✨ Возможности

### 🛡️ Права администратора (UAC)
- Автоматический запрос прав при запуске
- Двойная система: UAC манифест для `.exe` + `ShellExecuteW` для `.py`
- Отображение статуса прав в интерфейсе

### 🖥️ Надёжное создание ярлыков
- **3 метода создания ярлыков** с автоматическим фолбэком:
  1. COM-объект (`win32com.client`) — самый быстрый
  2. PowerShell — работает без дополнительных библиотек
  3. VBScript — запасной вариант
- Ярлык на **рабочем столе**
- Запись в **меню «Пуск»**
- Автоматическая иконка из главного файла

### 🎨 Красивый GUI (CustomTkinter)
- 5 страниц установщика: Приветствие → Путь → Опции → Установка → Готово
- Тёмная тема по умолчанию
- Анимированный прогресс-бар
- Детальный лог установки
- Выбор компонентов для установки

### 📦 Что внутри установщика
- Распаковка файлов из сжатого архива (base64 + zip)
- Выбор пользователем папки установки (по умолчанию `Program Files`)
- Проверка свободного места
- Создание `install_info.txt` с данными об установке
- Запуск программы после завершения

## 🔧 Как использовать

### 1. Установка зависимостей

```bash
pip install customtkinter pyinstaller pywin32 winshell
```

### 2. Запуск создателя

```bash
python creator.py
```

### 3. Ответьте на вопросы

```
▸ Название приложения [MyApp]: MyCoolApp
▸ Версия [1.0.0]: 2.5.0
▸ Автор [Developer]: YourName
▸ Описание: Отличное приложение!
▸ Папка с файлами: C:\Projects\myapp\dist
▸ Главный файл для запуска: myapp.exe
▸ Путь к .ico файлу: C:\icon.ico
▸ Скомпилировать в .exe? [y]: y
```

### 4. Готово! 🎉

Установщик появится в папке `/output/`:
- `MyCoolApp_Setup.py` — скрипт
- `MyCoolApp_Setup.exe` — готовый установщик

## ⚙️ Технические детали

### Архитектура

```
creator.py (консоль)
    │
    ├── Сбор информации о проекте
    ├── Упаковка файлов в base64 .zip
    ├── Генерация installer_template.py с подстановкой данных
    └── Компиляция в .exe через PyInstaller
         │
         └── installer_template.py (исполняемый установщик)
              │
              ├── UAC: запрос прав администратора
              ├── GUI на CustomTkinter (5 страниц)
              ├── Распаковка файлов
              ├── Создание ярлыков (3 метода)
              └── Запуск программы
```

### Создание ярлыков (3 метода)

```python
# Методы пробуются по очереди до успешного
1. COM (win32com.client)    → ⚡ Быстро
2. PowerShell               → 🪟 Работает везде
3. VBScript                 → 🔧 Запасной
```

### Получение пути к рабочему столу (4 метода)

```python
1. Shell API (SHGetFolderPathW)
2. %USERPROFILE%\Desktop
3. winshell.desktop()
4. Реестр Windows
```
